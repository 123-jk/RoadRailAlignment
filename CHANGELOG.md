# Changelog

## v0.2.1 - 2026-06-14

### Changed

- `保留匝道曲率给下一段` 默认改为不勾选，避免智能基础线形模式下默认把下一段直线按上一段曲率绘制成缓和曲线。
- 智能基础线形模式仍会自动保留上一段匝道曲率候选；只有下一段确认为同方向曲线时，才删除上一段原出口缓和曲线，并按两段曲率重建衔接缓和、下一段曲线和下一段出口缓和到 0 曲率的缓和曲线。

### Fixed

- 修复下一段接入直线时，连续匝道曲率复用逻辑误触发，导致直线被绘制为缓和曲线的问题。
- 修复关闭默认保留曲率选项后，智能基础线形模式下同方向连续曲线自动衔接逻辑不再生效的问题。

## v0.2.0 - 2026-06-14

本版重整匝道连续绘制流程，把接出匝道、创建新点和连接既有线合并进默认基础模式，并修正连续匝道曲率衔接时的几何与数据清理问题。

### Added

- 新增默认 `智能基础线形` 模式，可在同一模式内创建新点、从既有线接出匝道、连接两条既有线。
- 新增 `Keep ramp curvature for next segment` 选项，中文资源为 `保留匝道曲率给下一段`。
- 单端匝道默认在每段末尾追加缓和到 0 曲率的出口缓和曲线。

### Changed

- 当连续匝道下一段同向弯曲且开启保留曲率时，会废弃上一段末尾缓和到 0 的出口缓和曲线，改为用上一段末端带符号曲率衔接下一段。
- 匝道连续模式在上一段顺延时保持方向，支持回转匝道连续绘制。
- 生成要素不再写入插件 `source` 标识标签。
- 重写中文 README，并补充英文 README。

### Fixed

- 修复单端匝道出口缓和到 0 时未正确衔接圆曲线的问题。
- 修复废弃上一段出口缓和曲线时只更新 Way、未删除废弃节点的问题。

## 0.1.21 - 2026-05-24

本版把连续匝道、多语种 I18n 和预设标签一起整理完成，作为下一版发版。

### Added

- 增加高速公路、高速公路连接线、干线公路、一级公路、高速铁路等预设标签。
- 补齐中 / 英 / 法 / 德 / 西五语 I18n，并生成 JOSM `.lang` 资源。

### Fixed

- 修复单端匝道连续作业时，下一段没有继承上一段终点作为新起点的问题。
- 修复单端匝道连续作业时，不应再估算前一段曲率的问题。

## 0.1.20 - 2026-05-20

GitHub 首次发布整理版。该版本合并了 0.1.1 到 0.1.20 的功能积累，作为可试用 V1 发布。

### Added

- 增加 JOSM 插件入口、地图模式按钮、右侧参数面板和独立浮动参数窗口。
- 支持道路 / 铁路对象类型选择，并生成普通 JOSM `Node` / `Way`。
- 支持两点直线、三点 PI 圆曲线、带缓和曲线的 PI 圆曲线。
- 增加大角度圆曲线 / 回环模式，可绘制超过 180 度的弯道并支持附加整圈。
- 支持按当前半径推荐缓和曲线长度。
- 支持在插件模式内直接点击既有 Way，识别接入点、切线、近似里程、点击偏移、局部半径和曲率。
- 支持从一条既有 Way 切线接出单端匝道。
- 支持连接两条既有 Way，生成两端 G1 连续连接线。
- 两线连接支持自动最大半径 / 缓和曲线优化，并可选择是否回填优化结果。
- 两线连接增加“圆弧-直线-圆弧”和“三段圆弧”候选。
- 两线连接可按附加整圈数强制加入回环。
- 增加接入方向模式，支持自动朝目标、按所在线方向、反向所在线方向。
- 单端既有线匝道生成后支持继续作业，保留末端切线用于后续线形。
- 支持连续作业，直线和圆曲线生成后可自动保留终点作为下一段起点。
- 支持既有节点吸附和生成 Way 时复用被吸附节点。
- 增加插件内撤回一步，支持撤回最后一个控制点并同步恢复匝道 / 连续状态。
- 增加 `Esc` 清空控制点、预览和连续绘制状态。
- 增加几何烟雾测试脚本和 GitHub Actions 构建流程。

### Changed

- 将插件界面改为中文。
- 将参数控件抽成独立 `AlignmentPanel`，便于右侧面板和浮动窗口复用。
- 从主模式下拉框隐藏单独的缓和曲线近似片段模式，优先通过三点圆曲线或匝道模式启用缓和曲线。
- 三点圆曲线在连续模式下只生成到曲线终点，保留出切线方向交给下一段继续绘制。
- 默认保留手动半径和缓和长度；自动优化结果需要打开回填开关才写回参数栏。

### Fixed

- 修正 JOSM 19555 中右侧面板图标路径导致的加载崩溃。
- 修正三点圆曲线转角计算，使用实际转角而不是交点内角。
- 修正生成端点吸附复用逻辑，避免中间采样点被附近无关节点拉歪。
- 修正两节点既有线参与源线方向吸附时的局部曲率估算崩溃。
- 修正曲线段切出匝道时缓和段可能先向错误一侧转向的问题。
- 修正 PI 缓和圆曲线拼接处局部曲率过大的问题。
- 限制匝道缓和段接入曲率不超过设计最小半径。
